## How To Open

Open the `TARKSANGAM6` folder in VS Code.

To preview:

1. Open `index.html`.
2. Right-click inside the file.
3. Select `Open with Live Server`.

The registration form prepares a WhatsApp message for:

```text
+91 6200902995
```

WhatsApp will open with the candidate details filled in. The user still needs to press send.
